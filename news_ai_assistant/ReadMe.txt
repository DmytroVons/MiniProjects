**Setting up the News AI Assistant**

To set up the News AI Assistant, you'll need to utilize the `config.json` file with the provided valid configuration below.

**Valid Category Types:**
- sport
- world
- business
- science_and_technology
- education
- entertainment
- health
- travel

**Valid Countries:**
- us (United States)
- gb (United Kingdom)
- ng (Nigeria)
- eg (Egypt)
- sa (Saudi Arabia)
- ae (United Arab Emirates)
- ca (Canada)
- mx (Mexico)
- it (Italy)
- de (Germany)
- ar (Argentina)
- es (Spain)
- jp (Japan)
- au (Australia)
- fr (France)
- be (Belgium)
- tr (Turkey)
- ua (Ukraine)
- 144 (Russia)
- br (Brazil)
- id (Indonesia)
- in (India)

**Valid Languages:**
- en (English)
- ar (Arabic)
- fr (French)
- es (Spanish)
- it (Italian)
- de (German)
- ja (Japanese)
- nl (Dutch)
- ru (Russian)
- pt (Portuguese)
- id (Indonesian)

For the 'page' parameter, kindly specify the desired number of pages you wish to obtain.

Regarding the 'limit' parameter, please indicate the number of news articles you'd like to have on a single page.
